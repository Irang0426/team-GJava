## 1.User

-입력값의 길이는 3

-숫자 외의 값은 들어오면 안됨.

-자릿수가 중복되면 안됨. 

## 2.Computer

-3자리(1~9) 랜덤값 생성

-각 자리마다 숫자는 다름 

## 3.BaseballGame

-게임이 이루어짐

-user 와 computer 있어야함. 

-유저에게 입력값 받음 

-컴퓨터 랜덤값과 비교 

-스트라이크,볼,체크 

-현재 상황 출력

-모두 맞출 때 까지 위의 내용을 반복 

-다시 게임할 것인지 물어봄

-다시 하는거면 볼,스트라이크 초기화
